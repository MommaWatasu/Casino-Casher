このデコの概要

曰く付きのカジノの存在を知らされた「あなた」は捜査員としてカジノに潜入する。信じられるのは自分だけ。あなたはディーラーとの対決に勝利し、真偽を暴けるのか？

First Game~数当て~

最初に、チップをベットしていただきます。

次に、先手と後手を決めます。決めるのには以下のゲームを使います。
エースからKまで(1~13)の13種類の数字の中から、山札の一番上のカードの数字をあてていただきます。
当てることができた方が、先手後手を選択することができます。
共に外した場合、あてた場合はもう一度行います。
2回行っても決まらなかった場合は、プレイヤー側が選択できます。

先手後手を決めたら、1~5のカードがそれぞれ一枚ずつ渡されます。
先手と後手は交互にカードを1枚ずつ出していきます。出す際はカードを裏にしてください。
双方が手札を出し切ってから、プレイヤー側絡みて左側からディーラーがカードをめくっていきます。カードの数字が大きいほうが勝ちで、同じ場合は引き分けです。 勝ったとき、1セット獲得します。引き分けの場合は両者ともにセットは入りません。5セットすべたが終わったとき、多くのセット数を取っている方がゲームの勝者となります。勝利した場合はかけたチップの2倍を獲得、引き分けの場合はチップはそのまま、敗北した場合う　 はすべて回収されてしまいます。

Second Game~ブラック・ジャック~

最初に、チップをベットしていただきます。

このゲームでは、カードの数字の和を21に近づけることを目指します。
ディーラーがカードを2枚配るので、更に1枚引くか、そのままかを決めてください。このときに21を超えるとバーストと言って、失格になってしまいます。失格になるとチップが回収されてしまいます。

全員がカードを引き終えたら、ディーラーは自分のカードが17以上になるまでカードを引き続けます。最後に、ディーラーより21に近いと勝ち、ディーラーと同じ数なら引き分け、ディーラーのほうが21に近いと負けとなります。
また、ディーラーがバーストしたときはバーストしていないプレイヤー全員が勝ちとなります。

勝ったプレイヤーはベットしたチップの2倍のチップを獲得できます。
負けたプレイヤーのチップは回収されてしまいます。

Third Game~オリジナルゲーム~

最後はオリジナルのゲームをしていただきます。更に、このゲームに挑戦するにはall-inをすることが必要です。

まずはジョーカーを覗いた52枚のカードをシャッフルして20枚配られます。このとき配られた手札の中から数字が同じ2枚を選んでセットします。
先手と後手をじゃんけんで決めて、残った手札から一枚ずつ出していきます。ここで、相手がセットした数字と同じ数字を出したら負けとなります。
また、互いの手札の残り枚数が2枚となった場合は、ディーラーの勝ちとなります。
